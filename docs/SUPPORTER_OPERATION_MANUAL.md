## 1. ログイン方法

### システムにログインする

1. ブラウザで以下のURLにアクセス
   - **本番**: https://booking.furniturehouse1.com/supporter/
   - **テスト用**: https://staging.furniturehouse1.com/supporter/

2. 管理者から受け取った **メールアドレス** と **パスワード** を入力
   - パスワードは8文字以上で、大文字・小文字・数字を含む必要があります

3. 「ログイン」ボタンをクリック

> **ログインできない場合**: パスワードを忘れた場合は、管理者に連絡してパスワードの再設定を依頼してください。

| タブ | 内容 |

## 3. タスクを確認する

### タスク一覧を見る

1. ダッシュボードの「本日」または「未完了」タブをタップ
2. 割り当てられたタスクが一覧表示されます

### タスクカードの見方

```
┌────────────────────────────────┐
│ チェックアウト後清掃    [割当済]  │
│ ───────────────────────────── │
│ 予定日: 1月21日（火）           │
│ チェックリスト: 0/8 完了        │
└────────────────────────────────┘
```

### タスクの種類

| 種類 | 内容 | タイミング |
|------|------|-----------|
| チェックイン前清掃 | ゲスト到着前の準備清掃 | チェックイン日の朝まで |
| チェックアウト後清掃 | ゲスト退去後の清掃 | チェックアウト日の午後 |

### ステータスの意味

| ステータス | 意味 |
|-----------|------|
| 割当済 | あなたに割り当てられた状態 |
| 作業中 | 作業を開始した状態 |
| 完了 | 作業が終わった状態 |

## 4. 清掃作業を行う

### 作業の流れ

```
タスク確認 → 作業開始 → チェックリスト完了 → 備品記録 → 作業完了
```

### 手順1: 作業を開始する

1. タスクをタップして詳細画面を開く
2. **「作業開始」** ボタンをタップ
3. 作業時間の計測が始まります

> **重要**: 現場に到着してから「作業開始」を押してください。

### 手順2: チェックリストを完了する

清掃中、以下の項目を1つずつチェックしていきます。

| チェック項目 |
|-------------|
| ベッドシーツ交換 |
| 枕カバー交換 |
| タオル交換 |
| 床掃除 |
| トイレ掃除 |
| キッチン掃除 |
| ゴミ回収 |
| 備品チェック |

- 各項目が終わったら、チェックボックスをタップ
- 進捗が「3/8 完了」のように表示されます

### 手順3: 使用した備品を記録する

清掃で使った備品を記録します。

1. **「備品を追加」** ボタンをタップ
2. 備品名を選択（例: バスタオル、シャンプー）
3. 使用した数量を入力
4. 複数の備品を使った場合は、繰り返し追加

### 手順4: メモを残す（任意）

気づいたことや報告事項があれば、メモ欄に記入します。

例：
- 「エアコンのリモコンの電池が切れかけています」
- 「浴室の排水口が少し詰まり気味です」

### 手順5: 作業を完了する

1. すべてのチェックリストが完了していることを確認
2. **「作業完了」** ボタンをタップ
3. 報酬が自動計算されます

> **注意**: 一度「完了」にすると元に戻せません。すべての作業が終わってから押してください。

## 5. 報酬を確認する

### 今月の報酬を見る

1. ダッシュボードの **「報酬」** タブをタップ
2. 以下の情報が表示されます

| 項目 | 内容 |
|------|------|
| 今月の稼働時間 | 今月作業した合計時間 |
| 今月の報酬（税込） | 今月の報酬合計額 |
| 支払い状況 | 支払い済み/未払い |

### 報酬の計算方法

```
報酬 = 作業時間 × 時給 + 交通費
```

例：2時間30分の作業、時給1,300円、交通費600円の場合
```
2.5時間 × 1,300円 + 600円 = 3,850円
```

### 報酬履歴を見る

「報酬」タブの下部に、タスクごとの報酬履歴が表示されます。

| 日付 | タスク種別 | 稼働時間 | 報酬 | 支払い |
|------|-----------|---------|------|--------|
| 1/20 | チェックアウト後 | 2.5時間 | ¥3,850 | 未払い |
| 1/18 | チェックイン前 | 1.5時間 | ¥2,550 | 支払済 |

## 6. 備品を管理する

### 在庫一覧を見る

1. ダッシュボードの **「備品管理」** タブをタップ
2. アメニティの在庫状況が表示されます

### 在庫ステータスの意味

| ステータス | 色 | 意味 |
|-----------|-----|------|
| 十分 | 緑 | 在庫に余裕あり |
| 要発注 | 黄 | 発注が必要 |
| 在庫切れ | 赤 | すぐに発注が必要 |

### 在庫を更新する

備品を使用したり、補充したりした場合：

1. 更新したい備品の **「編集」** ボタンをタップ
2. 現在の在庫数を入力
3. **「保存」** をタップ

### 新しい備品を追加する

1. **「備品を追加」** ボタンをタップ
2. 以下の情報を入力
   - 備品名（例: ハンドソープ）
   - 現在の在庫数
   - 単位（個、本、枚など）
   - 発注目安の数量
   - 購入URL（任意）
3. **「追加」** をタップ

### メンテナンス記録を見る

「備品管理」タブの下部に、設備のメンテナンス履歴があります。

| 設備 | 前回メンテ | 次回予定 | 状態 |
|------|-----------|---------|------|
| エアコン | 12/15 | 3/15 | 良好 |
| 給湯器 | 11/20 | 5/20 | 要確認 |

## 7. 困ったときは

### よくある質問

| 質問 | 回答 |
|------|------|
| 作業開始を押し忘れた | 管理者に連絡して、手動で時間を記録してもらってください |
| 間違えて完了にした | 管理者に連絡してください |
| パスワードを忘れた | 管理者に連絡して再設定してもらってください |
| タスクが表示されない | ブラウザを更新してください。それでも表示されない場合は管理者に確認 |

### 緊急時の対応

| 状況 | 対応 |
|------|------|
| 設備の故障を発見 | タスクのメモに記録し、管理者に連絡 |
| 備品が足りない | 在庫ステータスを「在庫切れ」に更新し、管理者に連絡 |
| ゲストがまだ滞在中 | 作業を中止し、管理者に連絡 |
| 鍵が開かない | 管理者に連絡 |

### 連絡先

- **管理者メール**: furniturehouse1@chladni.co.jp
- **緊急連絡先**: （管理者から共有された電話番号）
