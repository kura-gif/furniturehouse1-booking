## 1. ログイン方法

### 管理画面を開く

1. ブラウザで以下のURLにアクセス
   - **本番**: https://booking.furniturehouse1.com/admin/
   - **テスト用**: https://staging.furniturehouse1.com/admin/

2. 「Googleでログイン」ボタンをクリック

3. 管理者として登録されているGoogleアカウントでログイン

> **ログインできない場合**: 管理者として招待されていない可能性があります。既存の管理者に招待を依頼してください。

## 2. 毎日やること

### 朝のチェック（5〜10分）

管理画面のダッシュボードで、以下を確認します。

| 確認項目 | 場所 | やること |
|---------|------|---------|
| 審査待ち予約 | ダッシュボード上部 | 承認または却下する |
| 本日のチェックイン | ダッシュボード | 準備ができているか確認 |
| 本日のチェックアウト | ダッシュボード | 清掃タスクが割り当てられているか確認 |
| 新しいメッセージ | 左メニュー「メッセージ」 | 返信する |

## 3. 予約を管理する

### 新しい予約を承認する

1. ダッシュボードの「審査待ち」セクションを確認
2. 予約をクリックして詳細を確認
3. 問題なければ **「承認」** ボタンをクリック
4. 問題がある場合は **「却下」** ボタンをクリックし、理由を入力

> **重要**: 予約は **72時間以内** に承認または却下してください。期限を過ぎると自動キャンセルになります。

### 予約を確認する

- ダッシュボードの予約一覧から、見たい予約をクリック
- 予約詳細画面で以下が確認できます
  - ゲスト情報（名前、連絡先）
  - 宿泊日程
  - 料金・支払い状況
  - 選択されたオプション

### 予約ステータスの意味

| ステータス | 意味 |
|-----------|------|
| 審査待ち | 新しい予約。承認待ち |
| 確定 | 承認済み。宿泊予定 |
| 完了 | チェックアウト済み |
| キャンセル | ゲストまたは管理者がキャンセル |
| 却下 | 管理者が承認しなかった |

## 4. ゲストからのメッセージに返信する

1. 左メニューから **「メッセージ」** をクリック
2. 「進行中」タブに未対応のメッセージが表示されます
3. メッセージをクリックして会話を開く
4. 下部の入力欄に返信を入力し、**「送信」** ボタンをクリック
5. 対応が完了したら **「完了にする」** ボタンをクリック

> **ヒント**: 未読メッセージがある場合、メニューに赤いバッジが表示されます。

## 5. レビューを確認・承認する

ゲストが投稿したレビューは、管理者が承認するまで公開されません。

1. 左メニューから **「レビュー管理」** をクリック
2. 「承認待ち」タブを確認
3. レビュー内容を確認
   - 適切な内容であれば **「承認」** をクリック
   - 不適切な内容であれば **「却下」** をクリック

### 承認の基準

| 承認 | 却下 |
|------|------|
| 実際に宿泊したゲストのレビュー | 宿泊していない人のレビュー |
| 建設的なフィードバック | 誹謗中傷・個人攻撃 |
| 率直な感想（低評価でもOK） | 関係ない内容・スパム |

## 6. 清掃スタッフを管理する

### スタッフ一覧を見る

1. 左メニューから **「サポーター管理」** をクリック
2. 登録されているスタッフの一覧が表示されます

### 新しいスタッフを登録する

1. **「新規サポーター追加」** ボタンをクリック
2. 以下の情報を入力
   - メールアドレス
   - 初期パスワード（8文字以上、大文字・小文字・数字を含む）
   - 氏名
   - 電話番号
   - 時給
   - 交通費
3. **「登録」** ボタンをクリック

> 登録後、スタッフには入力したメールアドレスにログイン情報を別途お知らせください。

### スタッフ情報を編集する

1. 編集したいスタッフの行の **「編集」** ボタンをクリック
2. 情報を修正
3. **「保存」** ボタンをクリック

### スタッフを無効にする

退職などでスタッフを一時的に無効にする場合：

1. スタッフの行の **「編集」** ボタンをクリック
2. 「アクティブ」のチェックを外す
3. **「保存」** ボタンをクリック

> **注意**: 削除ではなく「無効」にすることで、過去の作業記録は保持されます。

## 7. 清掃タスクを割り当てる

予約が確定すると、清掃タスクが自動的に作成されます。

### タスク一覧を見る

1. 左メニューから **「清掃タスク」** をクリック
2. タスクの一覧が表示されます

### タスクの種類

| 種類 | タイミング |
|------|-----------|
| チェックイン前清掃 | チェックイン日の朝まで |
| チェックアウト後清掃 | チェックアウト日の午後 |

### スタッフを割り当てる

1. 「未割当」のタスクをクリック
2. **「サポーター割り当て」** ボタンをクリック
3. 担当するスタッフを選択
4. **「割り当て」** をクリック

> 割り当てると、スタッフに自動でメール通知が送られます。

### タスクの完了を確認する

1. タスクをクリックして詳細を開く
2. 以下が確認できます
   - 清掃チェックリストの完了状況
   - アップロードされた写真
   - 使用した備品
   - 作業時間

## 8. 月末にやること

### サポーター報酬を確認する

1. 左メニューから **「報酬計算」** をクリック
2. 年月を選択
3. サポーターごとの報酬が表示されます
   - タスク数
   - 作業時間
   - 報酬額（時給 × 時間 + 交通費）

### 報酬データをダウンロードする

1. **「CSVエクスポート」** ボタンをクリック
2. ダウンロードしたファイルを経理処理に使用

### 支払い済みにする

振込完了後：

1. 各タスクの **「支払い済みにする」** ボタンをクリック
2. または一括で **「すべて支払い済みにする」** をクリック

## 9. 困ったときは

### よくあるトラブル

| 症状 | 対処法 |
|------|--------|
| ログインできない | 招待されているか確認。管理者に連絡 |
| 予約が表示されない | ブラウザを更新（F5キー） |
| ボタンが反応しない | 少し待ってから再度クリック |
| エラーが表示される | 画面を更新して再試行。改善しない場合は開発者に連絡 |

### 緊急連絡先

- **システム開発者**: （開発担当者の連絡先をここに記載）
- **管理者メール**: furniturehouse1@chladni.co.jp

### 操作を間違えたとき

| 操作 | 取り消し方法 |
|------|-------------|
| 予約を誤って却下 | 開発者に連絡して復旧依頼 |
| スタッフを誤って削除 | 「無効」にしただけなら再度「アクティブ」に |
| 清掃タスクの割り当て変更 | 再度別のスタッフを割り当て直す |
